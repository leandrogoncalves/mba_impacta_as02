## Atividade 02 - Terraform


### Objetivo
---

Subir uma máquina virtual no Azure, AWS ou GCP instalando o MySQL e que esteja acessível no host da máquina na porta 3306, usando Terraform.

### Requisitos
---

* Terraform
* Microsoft Azure Cli


### Execução

* execute os comandos abaixo no terminal:
```
$ az login
$ terraform apply
```
